# Basics				
---

###True/False				


```
C#
    true
    false

Python
    True
    False		

PowerShell
    $true
    $false		

```

###Null				


```
C#
    null		

Python 
    None		

PowerShell
    $null		
```

### Declare and Assign				


```

C#
    var hello = "world";		

Python 
	hello = "world"		

PowerShell
	$hello = "word"		
```

### if				

```
C#	
	if ( true ) { … }
	if ( false ) { … }
		
Python 
 	if ( True ) :
    	# do nothing
	if ( False ) :
    	# do nothing		

PowerShell
	if ( $true ) { }
	if ( $false ) { }		
```


### test for null				

```
C# 3.0	
	if ( a == null ) { }		

Python or IronPython	
	if (a == None) :
    	    # do nothing		

PowerShell
	if ( $a –eq $null ) { }		
```
