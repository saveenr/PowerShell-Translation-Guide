
# Loops				
---

### For	

```
C#
	for (int i=0; i<10;i++) { }		


Python
    for i in xrange(10) :
       pass		


PowerShell
    foreach ($i in 0..9)  { }		
```

### For Each

```
C#
	var a = [1,7,8,10]
	foreach (var i in a) { }		

Python
    a = [1,7,8,10]
    for i in a:
       pass		

PowerShell
	$a = @(1,2,3)
	foreach ($i in $a) { }		
```

### While	


```
C#
	while (true) { }		

Python
	while (True) :
    	pass		

PowerShell
	while ($true) { }
```
